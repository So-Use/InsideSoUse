Les partenaires n'ont pas forcément d'équipe de développement et utilise souvent des plateformes du style WordPress

Intégration facile

  • API Rest conforme aux "standards"
  • Doc à jour
  • Clé API récupérable directement (via inscription en ligne)

Intégration moins facile

  • Restriction par IP
  • Docs obsolètes
  • API générées par des plugins
  • SOAP
  • OAuth1
  • Pas d'API => récupération d'un catalogue d'offre (xml, csv, json...)

Monitoring

  • changements impromptus (payloads, url)
  • Monitoring du hub (détection des erreurs, gestion des quotas, etc ...)
  • Important car c'est le coeur de la plateforme
  • Nécessaire de voir ce qu'il se passe
